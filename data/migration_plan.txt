Build container:

$ docker build -t language-map/data .

Try stuff out in container:

$ docker run -it --volume=$(pwd)/scripts:/usr/src/app --volume=/home/peter/Downloads/lm:/tmp language-map/data

Transform PUMS data within container via:

$ ./transform_pums_data < /tmp/psam_pusa.csv > /tmp/output.json

To build vector tiles:

$ ./build_tiles /tmp/output.json /tmp/puma_2020 /tmp/puma_states /tmp/tiles

Put the .pbf files produced in the correct place via:

$ cp -r --force ~/Downloads/lm/tiles/ app/static/

Tool to inspect .mbtiles files produced:

$ docker run -it -v $(pwd):/data -p 8080:80 klokantech/tileserver-gl ci-4iKuXcnbSq/pumas.mbtiles
