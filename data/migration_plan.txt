Build container:

$ docker build -t language-map/data .

Try stuff out in container:

$ docker run -it --volume=$(pwd)/scripts:/usr/src/app --volume=/home/peter/Downloads/psam_pus:/tmp language-map/data

Transform PUMS data within container via:

$ ./transform_pums_data < /tmp/psam_pusa.csv > /tmp/output.json

To create GeoJSON from shapefile:

$ ogr2ogr -f GeoJSON -t_srs crs:84 tl_2020_42_puma20.geojson tl_2020_42_puma20.shp
